Plotting tutorial in Octave

Example: 2d parametric plot
Plot the paremetric circle $x= \cos{t}, y = \sin{t}$, $0 \le t \le 2\pi$. 
t = linspace(0,2*pi,100);
plot(cos(t),sin(t), 'linewidth',2)
axis([-2 2 -2 2])
Animated 2d graph
Trace the parametric curve $x= t\cos{t}, y = t\sin{t}$, $0 \le t \le 6\pi$. 
n = 100; 
t = linspace(0,6*pi,n);
x = t.*cos(t);
y = t.*sin(t);

for i = 1:n
    plot(x([1:i]), y([1:i]), 'linewidth', 2);
    hold on
    
    quiver(0,0,x(i),y(i));
    
    axis([-20 20 -20 20])
    pause(0.1)  
    hold off
end
3d curves
Trace the curve $x= \cos{t}, y = \sin{t}, z = t$, $0 \le t \le 6\pi$. 
n = 100; 
t = linspace(0,6*pi,n);
x = cos(t);
y = sin(t);
z = t;

for i = 1:n
    plot3(x([1:i]), y([1:i]),z([1:i]), 'linewidth', 2);
    
    %hold on
    %quiver3(0,0,0,x(i),y(i),z(i));
    
    axis([-2 2 -2 2 0 20])
    pause(0.1)  
    hold off
end
A parametric surface: cylinder

% a cylinder
x=cos(t);
y=sin(t);
z=s;

surf(x,y,z,'EdgeColor','none')

Vector vields:2D
Plot the vector field $\mathbf{F}(x,y) = -y \,\mathbf{i} + x \,\mathbf{j}$ 
[x,y]=meshgrid(-3:0.2:3, -3:0.2:3);
quiver(x,y,-y,x,1)
